# Vertux Develop

Лендинг второго направления студии Vertux: внутренние бизнес-приложения,
клиентские порталы, Telegram-боты и защищённые AI-модули.

## Локальный запуск

```powershell
python -m http.server 8130
```

Открыть `http://127.0.0.1:8130`.

## Деплой

GitHub Pages публикует ветку `main`. Все зависимости и шрифты лежат локально,
внешних CDN нет.
