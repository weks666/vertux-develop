# Vertux Nexus — product site

Многостраничная статическая презентация Vertux Nexus для GitHub Pages.

## Страницы

- `index.html` — cinematic-презентация продукта.
- `updates.html` — changelog.
- `download.html` — Windows и Web Access.
- `account.html` — вход по приглашению (без отправки данных до подключения backend).

## Локальный запуск

Откройте `index.html` или запустите любой статический HTTP-сервер в каталоге проекта.

## Публикация

GitHub Pages публикует ветку `main` репозитория `weks666/vertux-develop`.

## Важно

- Публичного установщика Nexus пока нет; кнопка ведёт на запрос установки.
- Форма входа является безопасной оболочкой и не отправляет введённые данные.
- Клиентские названия и данные не используются на публичном сайте.
