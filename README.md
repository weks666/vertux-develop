# Vertux Nexus — продуктовая витрина

Одностраничная продуктовая презентация Vertux Nexus: кинематографичный кабинет клиента, Workspace, AI-виджет, Hybrid и режимы Local Node / Managed Cloud. Публичной формы Nexus Factory на сайте нет: она предназначена только для администратора Vertux.

## Локальный запуск

```powershell
python -m http.server 4180 --bind 127.0.0.1
```

Открыть `http://127.0.0.1:4180`.

Кнопка скачивания намеренно тестовая: публичного установщика пока нет. Все шрифты лежат локально в `assets/fonts/`, внешних CDN нет. В футере используется только правильный watermark «Сделано в студии Vertux»; подпись «Разработано командой Vertux» предназначена только для окна чат-виджета.

## Публикация

Текущий каталог совместим с GitHub Pages. Перед публичным обновлением нужно проверить mobile/desktop, scroll-сцены, меню, модальное окно скачивания, контакты и отсутствие горизонтального скролла.
